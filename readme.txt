== Description ==

Best newsletter subscriptions plugin. newsletter subscriptions is a WordPress plugin which is newsletter subscriptions. It export newsletter subscriptions data in CSV format. Use shortcode [SUBSCRIPTION_NEWSLETTER] in page or text widget to show newsletter subscriptions.

If you think, that you found a bug in our newsletter subscriptions plugin or have any question contact us at mohdhanifkhan1507@outlook.com.

== Installation ==

* Upload newsletter-subscriptions folder to the /wp-content/plugins/ directory
* Activate the plugin through the Plugins menu in WordPress
* Now you are done. Enjoy the plugin!!!
* Add shortcode [SUBSCRIPTION_NEWSLETTER] in page or text widget to show newsletter subscriptions.
